DESTINATION IMAGES SETUP GUIDE
================================

IMPORTANT: This file explains how to set up images for each destination.

STRUCTURE:
----------
Each destination folder should contain:
- main.jpg (or main.png) - The main hero image
- gallery-1.jpg, gallery-2.jpg, etc. - Gallery images

QUICK START:
------------
1. For each destination, place your images in the corresponding folder
2. Name the main image as "main.jpg"
3. Name gallery images as "gallery-1.jpg", "gallery-2.jpg", etc.
4. The JSON file (Public/destinations.json) will automatically reference these paths

EXAMPLE:
--------
To update Thailand images:
1. Go to: Public/images/destinations/thailand/
2. Replace main.jpg with your Thailand hero image
3. Replace gallery-1.jpg, gallery-2.jpg, etc. with your gallery images
4. Refresh the browser to see changes

NOTE: The destinations.json file uses paths like:
"/images/destinations/thailand/main.jpg"
These paths are relative to the Public folder.
